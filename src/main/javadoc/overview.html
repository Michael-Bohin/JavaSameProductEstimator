JavaSameProductEstimator

Welcome to the documentation for the JavaSameProductEstimator project.

Project Overview

The JavaSameProductEstimator is a tool designed to estimate whether two products are the same based on various criteria. It includes adapters for different e-commerce platforms, normalizes data, and applies various algorithms to find similarities between products.

Modules

  • Product Parser: Parses product data from different sources. The main part of the code handles normalizing different json schemas using adapters for specific stores.
  • Same Product Estimator: Contains algorithms to estimate product similarity.
  • Configuration: Manages runtime configurations for the application.
